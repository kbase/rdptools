NAME

Bio::KBase::RDPTools::Client

DESCRIPTION

This module provides methods for the classifying and matching of DNA sequences.

classifySeqs

  $detailed_results, $hier_results = $obj->classifySeqs($seqs, $options)
Parameter and return types
$seqs is a reference to a list where each element is a Sequence
$options is a reference to a list where each element is a string
$detailed_results is a Handle
$hier_results is a Handle
Sequence is a reference to a hash where the following keys are defined:
	seqid has a value which is a string
	bases has a value which is a string
Handle is a reference to a hash where the following keys are defined:
	file_name has a value which is a string
	id has a value which is a string
	type has a value which is a string
	url has a value which is a string
	remote_md5 has a value which is a string
	remote_sha1 has a value which is a string

Description

Takes as input a list of options (namely, the k nearest neighbors and the minimum sab score), the file containing reference sequences, and the file containing query sequences. It returns a list of k results per query sequence with the matching sequences.

TYPES

Sequence

Description

A structure of representing DNA sequences.

string seqid

This parameter is the sequence ID. The format is an 'S' followed by 9 numbers.

string bases

This parameter represents the bases in the sequence. Possible values are any combination of 'A', 'C', 'T', 'G' or their lowercase equivalents.

Definition
a reference to a hash where the following keys are defined:
seqid has a value which is a string
bases has a value which is a string
